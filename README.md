# PIE Framework Website

This is the public site and documentation site for the PIE Framework.

It uses VuePress https://vuepress.vuejs.org

| cmd                    | description                    |
| ---------------------- | ------------------------------ |
| `yarn install`         | install                        |
| `yarn dev`             | dev server on localhost:8080   |
| `yarn build`           | build                          |
